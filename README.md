# travel-planner

个性化旅游攻略自动规划 Skill，支持郴州 / 长沙 / 衡阳 / 上海四城，含大众点评美食推荐 + 高德地图导航 + 天气预报。

## 功能特性

- 🗺️ **景点规划**：内置景点库（评分/门票/标签/建议游玩时长），自动生成每日行程
- 🍽️ **美食推荐**：集成大众点评 AI 搜索，实时查询早/午/晚餐推荐（含评分+人均价格）
- 🧭 **高德导航**：每日景点间驾车/公交/步行路线，含 ETA 和距离
- 🌤️ **天气预报**：实时天气 + 未来3天 + 穿衣/紫外线/舒适度等生活指数
- 📊 **多预算支持**：穷游 / 舒适 / 轻奢三档预算推荐

## 支持城市

| 城市 | 景点数量 | 特色 |
|------|---------|------|
| 郴州 | 10个 | 东江湖(雾漫小东江)、高椅岭、仰天湖草原、莽山 |
| 长沙 | 12个 | 橘子洲、岳麓山、湖南博物院、太平老街 |
| 衡阳 | 10个 | 南岳衡山、南岳大庙、石鼓书院 |
| 上海 | 10个 | 外滩、豫园、迪士尼、朱家角 |

## 快速使用

在 Dify 中使用 mini_claw 插件，上传本仓库打包为 zip 格式即可安装。

> 美食推荐需要配置 `DIANPING_TOKEN` 环境变量（可选，不配置时使用静态推荐）。

## 环境变量

| 变量名 | 说明 | 必填 |
|--------|------|------|
| `AMAP_KEY` | 高德地图 Web API Key | 是 |
| `DIANPING_TOKEN` | 大众点评 API Token | 否 |
