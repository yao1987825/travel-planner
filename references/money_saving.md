# 旅游全链路省钱攻略

> 核心原则：**省钱不是不花钱，而是把有限的预算用在刀刃上，确保每一天都能真正出发、真正玩到！**
> 
> ⚠️ 最常见的旅游"亏损"模式：过度省钱 → 行程安排太满/太空 → 当天体力不支或时间不够 → 很多地方没去成！

---

## 一、预算分配黄金法则

### "4321"分配原则

| 类别 | 占比 | 说明 |
|------|------|------|
| 🏨 住宿 | 30% | 不用最好，但要交通方便（地铁沿线），省出来的时间值钱 |
| 🎟 门票+活动 | 25% | 优先把钱花在"必去"景点，免费景点填充剩余时间 |
| 🍜 餐饮 | 25% | 正餐去本地馆子，早餐/午餐便利店+街边，晚餐可稍微犒劳自己 |
| 🚇 交通 | 10% | 城内全程地铁，长途高铁/机票提前购，景区间打车 |
| 🛍 购物/备用 | 10% | 严格预留；旅游纪念品能不买就不买，把钱省给下次旅行 |

## 二、时间规划防"踩空"原则

### 每日行程密度建议

| 体力等级 | 每天景点数 | 每日有效游览时间 | 说明 |
|---------|-----------|----------------|------|
| 高（high） | 4个 | 12h（09:00-21:00） | 适合年轻人，身体好，爱走路 |
| 中（mid） | 3个 | 10h（09:00-19:00） | 大多数人的标准 |
| 低（low） | 2个 | 8h（10:00-18:00） | 老人、带小孩、体力不佳 |

### ⚠️ "省钱省到没玩到"的典型错误

1. **每天塞4-5个景点**：路上耗时+排队时间根本不够，最后只走马观花，哪里都没玩透
2. **过度依赖远郊景点**：一来一回2-3小时，真正游览时间反而少；行程短时直接跳过
3. **高峰时段不买提前票**：节假日到场无票 → 空手而归 → 钱没省到，时间白费
4. **住在城郊省住宿**：每天多花1-2小时通勤，累死人还浪费时间

### 时间节点对应景点数量参考

| 旅行天数 | 合理景点总数（中等体力） | 说明 |
|---------|----------------------|------|
| 1天 | 2-3个 | 宁少勿多，选最想去的 |
| 2天 | 5-6个 | 每天2-3个，留半天自由活动 |
| 3天 | 8-9个 | 每天3个，最后半天逛市集 |
| 4天 | 10-12个 | 可加一个郊区景点 |
| 5天+ | 12-15个 | 可考虑跨区域联游 |

## 三、门票省钱策略

### 1. 提前网购必省的场景

| 平台 | 适合买 | 省钱幅度 |
|------|--------|---------|
| 美团/大众点评 | 餐厅团购、门票、演出 | 5-30% |
| 携程/飞猪 | 景区门票、机票、酒店套餐 | 10-20% |
| 景区官方微信公众号 | 预约票、早鸟票 | 0-15%，但必须提前 |
| 拼多多 | 部分景区门票（需核实真伪） | 可能最便宜，但注意退款政策 |

### 2. 身份优惠记得带证件

| 身份 | 适用优惠 | 注意 |
|------|---------|------|
| 学生（含研究生） | 半价（部分免费） | 需学生证，部分需学信网截图 |
| 60-64岁老人 | 半价 | 需身份证 |
| 65岁以上 | 免费（多数景区） | 需身份证 |
| 军人/残疾人 | 免费 | 需有效证件 |
| 未成年（18岁以下） | 半价或免费（多数） | 需身份证 |

## 五、住宿省钱策略

### 选址原则（比价格更重要！）

> 住贵一点但住在地铁沿线，比住便宜但每天打车贵10倍更划算。

| 住宿类型 | 适合人群 | 省钱技巧 |
|---------|---------|---------|
| 青年旅舍（床位） | 独自旅行、学生 | 约50-80元/晚，社交氛围好 |
| 经济酒店（如如家、锦江） | 情侣、小家庭 | 约150-250元/晚，提前预订省20-30% |
| 民宿（闲鱼、民宿APP） | 家庭、长住 | 比同等酒店便宜15-30%，但注意退款政策 |
| 连锁品牌酒店 | 商务/舒适 | 会员积分+非节假日入住最划算 |

### 各城推荐住宿区域（地铁方便）

| 城市 | 推荐住宿区域 | 原因 |
|------|------------|------|
| 郴州 | 北湖区市中心 / 苏仙岭周边 | 景点集中，出行方便 |
| 长沙 | 五一商圈 / 岳麓区 | 美食集中，地铁发达 |
| 广州 | 越秀区北京路周边 / 天河区 | 景点密集，地铁四通八达 |
| 上海 | 人民广场 / 外滩周边 / 静安区 | 步行可达多个核心景点 |

## 六、交通省钱策略

### 城内交通

| 方式 | 省钱建议 |
|------|---------|
| 地铁 | 首选，广州/上海/长沙地铁均价2-5元；开通互联互通卡享9折 |
| 公交 | 部分城市老人免费、学生半价 |
| 共享单车 | 最后1公里神器，月卡约15-25元 |
| 网约车 | 滴滴/高德拼车比快车便宜30-40%；多人出行直接用快车摊薄成本 |
| 景区免费摆渡 | 很多景区内部免费接驳，不要被导游忽悠坐收费观光车 |

### 城际交通

| 线路 | 高铁时长 | 提前购票建议 |
|------|---------|------------|
| 郴州→长沙 | 约1.5h | 提前3天，约50-80元二等座 |
| 长沙→广州 | 约2.5h | 提前7天，约130-180元二等座 |
| 广州→上海 | 约7-8h | 建议搭飞机（约300-600元），提前1-2个月购 |
| 长沙→上海 | 约5.5h | 约350-450元二等座，提前购更便宜 |
